Update the Customize button to be clearer about what it does.

Find the Customize button (around line 792-797):
```rust
button {
    class: "btn btn-customize",
    onclick: move |_| settings_open.set(!settings_open()),
    i { class: "fa-solid fa-screwdriver-wrench" }
    span { " Customize" }
}
```

Change to add a tooltip and optionally adjust the label:
```rust
button {
    class: "btn btn-customize",
    title: "Open overlay appearance and behavior settings",
    onclick: move |_| settings_open.set(!settings_open()),
    i { class: "fa-solid fa-screwdriver-wrench" }
    span { " Settings" }
}
```

Per CONTEXT.md, the decision was to either change to "Customize Overlays" or add a title attribute. Using "Settings" with a tooltip is cleaner and more conventional.
